## 🎮 Возможности

- **Inline-режим** — не нужно добавлять бота в чат, просто вызовите через `@username_бота`
- **Разделение по чатам** — данные хранятся отдельно для каждой группы
- **Дуэли** — вызывайте участников на дуэль с кнопками ставок
- **Писюн дня** — случайный участник получает бонус каждый день
- **Топ участников** — рейтинг самых больших биб в чате

## 📦 Установка

### 1. Создайте бота в Telegram

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям: укажите имя бота и username (должен заканчиваться на `bot`)
4. BotFather выдаст токен вида `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`

### 2. Включите Inline Mode

1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте `/mybots` и выберите своего бота
3. Нажмите **Bot Settings** → **Inline Mode**
4. Нажмите **Turn on**

### 3. Клонируйте репозиторий

```bash
git clone <url-репозитория>
cd tg_bot
```

### 4. Настройте переменные окружения

Создайте файл `.env` на основе шаблона:

```bash
cp .env.example .env
```

Откройте `.env` и вставьте свой токен:

```
BOT_TOKEN=ваш_токен_от_botfather
BOT_USERNAME=@FairDickGrowerBot
```

> ⚠️ **Важно:** Файл `.env` добавлен в `.gitignore` и не попадёт в репозиторий.

### 5. Установите зависимости

```bash
python -m pip install -r requirements.txt
```

## 🖥️ Запуск на сервере

### Локальный запуск

```bash
python main.py
```

### Запуск в фоне (systemd)

Создайте файл `/etc/systemd/system/fairdickbot.service`:

```ini
[Unit]
Description=FairDickGrowerBot Telegram Bot
After=network.target

[Service]
Type=simple
User=ubuntu
WorkingDirectory=/path/to/tg_bot
ExecStart=/path/to/python main.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

Запустите сервис:

```bash
sudo systemctl daemon-reload
sudo systemctl enable fairdickbot
sudo systemctl start fairdickbot
sudo systemctl status fairdickbot
```

### Запуск через screen/tmux

```bash
screen -S fairdickbot
python main.py
# Ctrl+A, D — отсоединиться от сессии
```

## 🎯 Использование

### Inline режим (основной способ)

1. В любом чате (группе, супергруппе или личных сообщениях) начните вводить `@username_бота`
2. Выберите команду из списка кнопок
3. Нажмите кнопку в появившемся результате
4. Результат появится в чате с пометкой "via @username_бота"

**Пример:**
- Напишите `@FairDickGrowerBot` → появятся кнопки: Grow, Top, Dick Of Day, Stats, Duel
- Нажмите 📈 Grow → бибa вырастет на случайное значение

## 📖 Описание команд

#### 📈 Grow
Увеличивает вашу бибу на случайное значение от **-10 до +35 см**.

- Можно использовать **1 раз в сутки** после 00:00 по МСК
- Результат сохраняется и накапливается
- Значение может быть отрицательным — берегись!

#### 🏆 Top
Показывает топ-10 участников чата, которые хоть раз использовали `/grow`.

- Отсортировано по убыванию размера
- Показывает имя участника (с @username или ссылкой) и текущий размер в см

#### 🎉 Dick Of Day
Выбирает случайного участника чата и даёт ему бонус от **+1 до +30 см**.

- Можно использовать **1 раз в сутки**
- Бонус всегда положительный
- Участвуют все, кто хоть раз вызывал `/grow`
- При повторном вызове показывает, кто уже выбран сегодня

#### 📊 Stats
Показывает личную статистику дуэлей:

- ✅ Количество побед
- ❌ Количество поражений
- 📈 Суммарно выигранных см (может быть отрицательным)

#### ⚔️ Duel
Создаёт вызов для других участников чата.

- Доступные ставки: **1, 2, 3, 4, 5 см** (только те, что ≤ вашего текущего размера)
- Любой участник может принять вызов кнопкой
- Исход определяется случайно (50/50)
- Победитель получает ставку, проигравший теряет
- Если у участника отрицательный баланс — он не может участвовать

**Пример:**
1. Нажмите `@FairDickGrowerBot duel`
2. Выберите ставку из кнопок (1, 2, 3, 4 или 5 см)
3. Бот создаст вызов с кнопкой "Принять вызов!"
4. Другой участник может нажать кнопку и вступить в дуэль

## 📁 Структура проекта

```
tg_bot/
├── main.py           # Точка входа, регистрация обработчиков
├── handlers.py       # Обработчики inline-запросов и кнопок
├── database.py       # Работа с SQLite (создание таблиц, запросы)
├── config.py         # Настройки и константы бота
├── requirements.txt  # Python-зависимости
├── .env              # Переменные окружения (токен) — не в git!
├── .env.example      # Шаблон для .env
├── .gitignore        # Исключения для git
└── fairdick.db       # База данных (создаётся автоматически)
```
